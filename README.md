# 💰 จดเงิน (Jod Ngern)

> A lightweight, offline-first personal finance tracker — built as a single HTML file with no dependencies.

---

## Features

- **Track income & expenses** with emoji categories
- **Daily budget** with a visual progress bar and overspend alert
- **Spending streak** — stay motivated by logging every day 🔥
- **Monthly cycle** — set your own payday start date
- **Quick-add** shortcuts for frequent transactions
- **Savings goals** with progress tracking
- **Badges** — unlock achievements as you build better habits
- **Backup & restore** — export/import your data as JSON
- **PWA-ready** — install to your home screen, works offline

---

## Getting Started

No installation needed. Just open `index.html` in any modern browser.

To use as a home screen app on mobile:

1. Host the file on any static server (GitHub Pages, Netlify, etc.)
2. Open the URL in Safari (iOS) or Chrome (Android)
3. Tap **Share → Add to Home Screen** (iOS) or use the install banner (Android)

---

## Tech Stack

- Vanilla HTML, CSS, JavaScript — zero dependencies
- `localStorage` for persistent data storage
- Service Worker for offline support
- Web App Manifest for PWA installation

---

## Data & Privacy

All data is stored locally in your browser. Nothing is sent to any server.
Use the **Backup** feature in Settings to export your data as a `.json` file.
